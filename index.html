First Sight

First Sight is een dating app voor blinden en slechtzienden. Vooraf aan het bouwen heb ik de doelgroep goed leren kennen door middel van een eigengeschreven, goed doordacht onderszoeksplan. Uit dit onderzoek is een design specification gekomen en aan de hand daarvan heb ik deze applicatie gebouwd.

Datavisualisatie D3

Sinds de komst van de Millenial generatie op de werkvloer is het aantal "burnout" gevallen drastisch gestegen. Ik was benieuwd of deze stijging ook terug te vinden is in het aantal publicaties over het hebben van een burnout. Vanuit de Hva kreeg ik toegang tot de API van de Oba. Met een dataset uit deze API heb ik met behulp van D3 een datavisualisatie gebouwd.
